# MapCensorApp

> 基于webpack-vue2.0-mintUI 项目

## 打包安装

``` bash
# 安装依赖
npm install
 
# 启动服务 localhost:8080
npm run dev
 
# 打包压缩项目
npm run build
 
# 单元测试
npm run unit
 
# 自动化e2e测试
npm run e2e
 
# 单元测试 + e2e (npm run unit + npm run e2e)
npm test
```
