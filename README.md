# MapCensorApp

> 基于webpack-vue2.0-mintUI 项目

## 打包安装

``` bash
# 安装依赖
npm install

# 启动服务 localhost:8080
npm run dev

# 打包压缩项目
npm run build

# 单元测试
npm run unit
```

## 注意事项

> 1、目前没加过滤器，登录认证有些问题，点击退出可能无法完全退出，需要重新刷新；
>
> 2、当前只支持权限为superManage的待审核；
